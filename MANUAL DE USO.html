Manual de uso do sistema de automação dos relatórios de termografia Predimec®️

Sumário

  1. Adicionar arquivos

  2. Execução da aplicação

  3. Arquivos formatados


1. Adicionar arquivos

Para adicionar os arquivos que serão formatados, primeiro tem que ter em mente que apenas dois arquivos por vez devem ser adicionados, sendo eles um .docx (arquivo WORD) e outro .xslx ou .xlsm (arquivo EXCEL).
Os dois arquivos devem ser colocados dentro da pasta raíz do prjeto. Assim que os dois arquivos forem adicionados, o programa já poderá ser executado.


2. Execução da aplicação

Depois de fazer os passos descritos no item 1. Adicionar Arquivos, você poderá executar o arquivo Predimec Termografia.exe dando inicio a execução do programa.
Após isso abrirá uma tela para você acompanhar o sistema formatando os dois arquivos. Depois de um tempo o sistema irá avisar que a formatação foi concluída, podendo já fechar a aplicação e obter os arquivos formatados.

OBS.: A aplicação também avisa caso algum erro acontecer, explicando-o para você consertá-lo.


3. Arquivos formatados

Para obter os arquivos formatados, basta ir na pasta raíz do projeto e abrir a pasta "ARQUIVOS FORMATADOS" onde vai estar os arquivos prontos.

OBS.: Caso esta pasta for excluída ou por algum caso não estiver na pasta raíz do projeto, o sistema irá criar a mesma automaticamente.
